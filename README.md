# Horse Health Dataset - Exploratory Data Analysis

## Project Overview

This project performs Exploratory Data Analysis (EDA) and data preprocessing on the Horse Health dataset. The objective is to understand the dataset, identify data quality issues, handle missing values, visualize important features, and prepare the dataset for future machine learning models.

The notebook follows a structured EDA workflow commonly used in real-world data science projects.

---

## Objectives

- Understand dataset structure
- Analyze missing values
- Remove highly incomplete features
- Perform statistical analysis
- Explore feature distributions
- Detect outliers
- Study relationships between variables
- Prepare clean data for machine learning

---

## Dataset

The dataset contains clinical information collected from horses to study survival outcomes.
-Dataset: 
1. Total Instances: 299
2. Total Features: 28
3. Categorical Features: 17
4. Numerical Features: 11 (7 float, 4 integer)
5. Missing Values: 1,602 across multiple features

## Workflow

1. Import required libraries
2. Load dataset
3. Inspect dataset dimensions
4. Generate descriptive statistics
5. Analyze data types
6. Check missing values
7. Calculate missing value percentages
8. Remove columns with excessive missing data
9. Handle remaining missing values
10. Perform exploratory visualizations
11. Prepare cleaned dataset
